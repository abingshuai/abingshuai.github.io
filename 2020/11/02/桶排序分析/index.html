桶排序分析

一.简介

​ 划分多个范围相同的区间，每个子区间自排序，最后合并

​ 桶排序适用于数据均匀分布，需要避免单个桶内出现过多元素。

二.排序步骤

  • 设置一定量的数组作为空桶

  • 遍历数组按照划定范围放入空桶中

  • 对每个非空桶进行排序

  • 把每个排好序的非空桶拼接

四.代码解析

// 计算桶的数量
int bucketNum = (max - min) / arr.length + 1;

此处桶的数量计算不适用一些极端情况，如最大最小差值100，数组长度10000，此处除数可为其他任意值，避免大量元素放入同一桶内即可

五.复杂度分析
