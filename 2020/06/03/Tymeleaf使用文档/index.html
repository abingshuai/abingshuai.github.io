1. Tymeleaf简介

1.1 什么是Tymeleaf

Tymeleaf是适用于web和独立环境的现代服务端Java模板引擎，能处理HTML，XML，JavaScript，CSS甚至纯文本。

Thymeleaf的主要目标是实现一种优雅且高度可维护的模板创建方式。为此，它以’自然模板‘的概念为基础，以不影响模板用作设计原型的方式将其逻辑注入模板文件。这样可以改善设计沟通，并缩小设计团队与开发团队之间的差距。

Thymeleaf的设计也从一开始就考虑到了Web标准-尤其是HTML5 -允许您在需要时创建完成验证的模板。

1.2 Thymeleaf可以处理哪种模板？

  • HTML

  • XML

  • TEXT

  • JAVASCRIPT

  • CSS

  • RAW

    有两种标记模板模式（HTML和XML），三个文本模板模式（TEXT，JAVASCRIPT和CSS）和无操作模板模式（RAW）。
